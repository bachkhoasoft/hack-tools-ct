Options Dialog

Все настройки программы находятся в диалоге Options.


Scan List

Здесь вы можете задать список адресов которые будут использовать сканеры (HostName Scanner, Ports Scanner, NB Scanner, Ping Scanner, LookUp) при включенной опции "From List". Кроме IP адресов вы можете задать диапазон адресов (через символ '-') а также имена хостов.

Пример :
;-----------------------------------------------------------------
199.66.10.12
133.65.12.250 - 133.65.13.17
200.32.12.188 - 200.32.12.95    ; текст после ';' игнорируется
200.32.12.122 - 200.32.12.25
199.66.10.12

www.chat.ru     ; для отдельного компьютера Вы можете указать его имя
www.garris.com - www.yahoo.com ; - ошибочная строка !
199.66.10.15
;-----------------------------------------------------------------
 

Ports

На этой странице Вы можете задать список портов для сканирования Ports Scanner'ом.

What is a Port ?
The TCP port number identifies a process or application inside the computer.
  Any application or process that uses TCP for its transport is assigned a unique identification number called a TCP port. These ports are numbered beginning with zero. Port numbers for client applications are dynamically assigned by the operating system when there is a request for service. Port numbers for server applications are preassigned by the IANA (Internet Assigned Numbers Authority) and do not change.
  A server application or process that uses TCP as a transport has at least one preassigned port number. For example, the preassigned port numbers for FTP server services are 20 (data) and 21 (control).
 

Ping/Trace
Тут задаются параметры для утилиты Trace и для всех утилит которые используют ping (кроме Hosts Monitor'a, так как в мониторе указиваются параметры отдельно для каждого контролируемого хоста).

Option "Packets to send"
определяет количество посылаемых ECHO пакетов

Option "Packet Size"
задает размер ECHO пакета

Option "TimeOut"
определяет timeout интервал в милисекундах

Option "Time to Live"
значение TTL для посылаемых пакетов

Option "Number of Hops"
specifies maximum number of hops to search for target

Option : "Show Hosts Description"
Начиная с 0.09 версии программа может выдавать информацию о доменах, хостах, etc. (страницы LookUp, Scanner, ScanPorts, Trace с включенной опцией Use DNS). Если эта опция включена то рядом с каждым именем хоста будет информация типа :
... www.chat.ru        (Russia)
... www.gazprom.ru     (Join-stock company GAZPROM, town Moscow, Russia)
... mainboss.xaxa.BD   (Bangladesh)
  
Advanced
На этой странице размещены настройки сканера портов и Shared resources Scanner

Options for ports Scanner :

  • GetHostName before scan ports
    если включить эту опцию то прежде чем сканировать порты программа попытается узнать имя машины по IP адресу, и если это не удастся то порты сканироватся не будут, программа перейдет к следующему адресу из списка

  • Ping before scan ports
    а если включить эту опцию то перед сканированием портов программа пропингует машину по IP адресу и если не дождется ответа то перейдет к следующему адресу
    Параметры пингования см. на странице Ping/Trace

  • Show remote service messages
    если поставить здесь галочку то программу не будет сразу после соединения рвать connect, а подождет что скажет host. Ждать будет указанное в следующем поле время.

  • Time for wait message
    Вот тут и указывается сколько времени после коннекта ждать сообщения от сервера.

  • Send string to remote host
    Кроме того программа может после соединения послать серверу заданную строку. Здесь она и задается.

Options for NB Scanner :

  • Ping before scan resources
    если включить эту опцию то перед сканированием выделенных ресурсов программа пропингует машину по IP адресу и если не дождется ответа то перейдет к следующему адресу.
    Параметры пингования см. на странице Ping/Trace

  • GetHostName before scan
    а если включить эту опцию то прежде чем сканировать ресурсы программа попытается узнать имя машины по IP адресу. Если это не удастся и опция пингования отключена то ресурсы сканироватся не будут, программа перейдет к следующему адресу из списка. А если определить имя машины не удалось, но пингование было включено и было успешным то программа все равно будет искать ресурсы.

 
IP-Monitor
Настройки IP-Monitor'а : интервал обновления информации, цвета фона и графиков, количество линий в масштабных сетках.
 
Interface
Здесь Вы можете изменить параметры интерфейса программы :

- палитра цветов (стандартная windows'ская, зеленое на черном, желтое на синем, голубое на синем)
- шрифт текста в окнах с информацией
- help language (Russian or English)

- option : "Auto save options of main window pages"
If this options enabled and you press [Start] button then program save parameters of current page. IP-Monitor save size of monitors too

- option : "Auto save current page"
Если включить эту опцию то программа будет запоминать активную страницу, и кроме того если включена опция "Auto save options of main window pages" и при выходе из программы был запущен "Hosts Monitor" то при повторном запуске программы монитор сразу станет активным.

- option : "Allow page multiline"
Если включить эту опцию то закладки страниц программы при необходимости будут размещатся в несколько строчек. А если не включить то закладки будут в одну строчку но с кнопочками для скролинга.

- option : "Clear old info"
если включить эту опцию то при нажатии на кнопку Start старое содержимое окна будет удалятся

- option : "Minimize to tray"
- option : "Always top"

- option : "Kept in lists recent addresses"
если включить эту опцию то программа будет запоминать введенные вами адреса в списках.
 
Sound
А на этой странице задаются звуковые эффекты для разных событий (завершение работы утилитами, изменение статуса контролируемого хоста, ...)
