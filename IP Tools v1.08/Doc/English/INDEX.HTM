IP-Tools offers many TCP/IP utilities in one program. This program can work under Windows 95, Windows 98, Windows NT 4.0 and Windows 2000 and is indispensable for anyone who uses the Internet or Intranet.

IP-Tools include 15 utilities :

Local Info - examine the local host and show info about processor, memory, Winsock data, etc
Connection Info - show information about current TCP/IP network connections
NetBIOS Info - get NetBIOS information about network interfaces (local and remote computers)
NB Scanner - shared resources scanner
HostName Scanner - scan all hostnames under diapason of addresses
Ports Scanner - scan host services  (support diapason of addresses, like 193.15.1.1 - 193.15.3.255)
Ping Scanner - ping a remote hosts over the network (support diapason of addresses)
Trace - trace the route to a remote host over the network
WhoIs - obtain information on names from the Network Information Center
Finger - finger one or more users at a remote host
LookUp - look up a domain names according to its IP address or an IP address from its domain name
GetTime - get time from time servers (and synchronize with local clock)
Telnet client - telnet client
IP-Monitor - show in real time graphics for TCP,UDP,ICMP In,Out,Error packets
Hosts Monitor - monitoring the up/down status of selected hosts.


Features :

  • Program allows multitask operation - You may use all utilities at the same time.

  • All scanners can obtain information from a single host, from all hosts in diapason IP addresses (ex. 195.128.74.1 - 195.130.200.5) or work with list of hosts and IP addresses.

  • All information which program show you may save to text files (some information you may save to HTML files).

  • Highly intuitive interface makes operation easy for users

  • Easy Install / Upgrade / UnInstall program

  • Online registration via Internet


System Requirements:

  • Microsoft Windows 95 / 98 / NT 4.0 or Windows 2000 (recommended Windows NT 4.0 with SP 3 or Windows 2000)
  • Microsoft TCP/IP protocol
