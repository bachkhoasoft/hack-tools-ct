Ports

On this page You can select TCP ports which will checked by PortsScanner.

What is a Port ?
The TCP port number identifies a process or application inside the computer.
  Any application or process that uses TCP for its transport is assigned a unique identification number called a TCP port. These ports are numbered beginning with zero. Port numbers for client applications are dynamically assigned by the operating system when there is a request for service. Port numbers for server applications are preassigned by the IANA (Internet Assigned Numbers Authority) and do not change.
  A server application or process that uses TCP as a transport has at least one preassigned port number. For example, the preassigned port numbers for FTP server services are 20 (data) and 21 (control).
 

Ping/Trace

On this page You may choose Packet Size, Timeout, Packets to send, TTL, number of hops.

Option "Packets to send"
specified the number of ECHO packets

Option "Packet Size"
specified size of ECHO packets

Option "TimeOut"
specifies a timeout interval in milliseconds

Option "Time to Live"
specifies a TTL value

Option "Number of Hops"
specifies maximum number of hops to search for target

Option "Show Host's Description"
Program may show host's description (on pages : LookUp, Scanner, ScanPorts, Trace with enabled options "Use DNS"). If Options "Show Host's Descriptions" enabled then program show host's description near host's name. Like this :
... www.chat.ru        (Russia)
... www.gazprom.ru     (Join-stock company GAZPROM, town Moscow, Russia)
... mainboss.xaxa.BD   (Bangladesh)
  
Advanced

On this page locate options for Ports Scanner, for Shared resources Scanner and for Connections list

Options for ports Scanner :

  • GetHostName before scan ports
    if this option enable then before scan ports program try resolve host name via host's IP address. If this action fail then program go to next host from the list.

  • Ping before scan ports
    if this option enable then before scan ports program try ping host. If host no reply then program go to next host from the list. Options for ping see on page Ping/Trace in Options dialog.

  • Show remote service messages
    if this option is disable then program after connect with host immediate disconnect. Otherwise program waits message from host (and write this message to log window).

  • Time for wait message
    in this field You can define time for wait message from hosts.

  • Send string to remote host
    Program can send string to host after connect. In this field You can define this string.

Options for NB Scanner :

  • Ping before scan resources
    if this option enable then before scan resources program try ping host. If host no reply then program go to next host from the list. Options for ping see on page Ping/Trace in Options dialog.

  • GetHostName before scan
    if this option enable then before scan resources program try resolve host name via host's IP address. If this action fail and previous option is disable then program go to next host from the list. If this action fail but previous option is enable and ping succeed then program however will scan host's resources.

Option for Connections list : Auto refresh every ... msec

Interface

On this page you may define next parameters :

- colors palette for output windows (standard, green on black, yellow on blue or light blue on blue)
- font for output windows
- help language (Russian or English)

- option : "Auto save options of main window pages"
If this options enabled and you press [Start] button then program save parameters of current page. IP-Monitor save size of monitors too

- option : "Auto save current page"
If this option enabled then program will remember current page and activate this page after load, otherwise program always start with active page "Local Info". Besides if this option enabled and enabled option "Auto save options of main window pages" then program remember status of Hosts Monitor, and after load may start Hosts Monitor.

- option : "Allow page multiline"
If this option enabled then tabs are displayed on more than one row when all tabs doesn't fits across the top of the tab control. If this option is False, the tabs are displayed on one row only, and you must scroll the displayed scroll arrows to view all the tabs.

- option : "Clear old info"
if this option is enabled and you press [Start] button in any IP-Tools utility then output window will clear

- option : "Minimize to tray"
- option : "Always top"
- option : "Kept in lists recent addresses"
